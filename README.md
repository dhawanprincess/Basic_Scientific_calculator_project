## Abstract
A **C++ console-based Scientific Calculator** performing basic arithmetic, scientific functions, memory operations, unit conversions, and number utilities like prime check, GCD/LCM, and Fibonacci series, with calculation history support.

---

## Description
This project demonstrates practical application of **C++ programming concepts** such as functions, loops, conditionals, recursion, vectors, and global variables. It serves as an educational tool for students to strengthen their understanding of mathematical computations and C++ fundamentals.

---

## Features

1. **Basic Arithmetic**
   - Addition (+), Subtraction (-), Multiplication (*), Division (/)
   
2. **Scientific Functions**
   - Sine, Cosine, Tangent (in degrees)
   - Logarithm (base 10)
   - Square root

3. **Memory Operations**
   - Memory Add (M+)
   - Memory Subtract (M-)
   - Memory Recall (MR)
   - Memory Clear (MC)
